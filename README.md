## Project Overview
This repository contains a series of tasks demonstrating core Git competencies, including:
* **Branch Management:** Creating and managing feature branches.
* **Conflict Resolution:** Manually resolving merge conflicts between competing branches.
* **Work Preservation:** Using `git stash` to manage interrupted workflows.
* **Advanced History:** Utilizing `cherry-pick` to apply specific commits.
